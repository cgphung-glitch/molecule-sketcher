Simple Hückel Molecular Orbital Analysis

π electrons:
Auto-detect finds atoms with a double/triple bond, plus any singly-bonded atom (like a pyrrole-type N or furan-type O) bridging two π atoms already in the set. It can't see conjugation that only exists via resonance structures (e.g. an allylic cation with no drawn double bond on the charged carbon) — use the selection tool to add/remove atoms by hand. π-electron count is auto-computed but editable.

Save structure file

.mol saves 2D coordinates by default (or 3D, if you've generated a 3D structure and confirm the prompt). .xyz requires a generated/loaded 3D structure. SMILES output is a simplified, non-aromatic writer.

Import structure

Importing replaces the current 2D drawing (a 2D layout is generated automatically) and clears the 3D view — use "Generate 3D" or "Look up in PubChem" afterward. Files with real 3D coordinates (most .mol/.sdf/.xyz from PubChem) populate the 3D view immediately.
